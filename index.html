How to Host a Live Site on GitHub

Zeph Wood

These are the steps in how to put your html website live online through GitHub Bash

Step One:

Right click on the folder that contains your website and select GitBash

Step Two:

Type the following commands into GitBash to setup

  • git init (enter)
  • git checkout -gh-pages (enter)
  • git status (enter)

    • Step Three:

    Type the following commands into GitBash to commit

  • git add filename
  • git status
  • git commit -m "add a message to decribe what you are uploading"
  • git push
  • git status

    • Step Four:

    Now you should setup a github repository

  • navigate to GitHub.com
  • select add a repository
  • name your repository

    • Step Five:

    Input the following commands into GitBash

  • copy and paste the first link "git remote add origin https://github.com/woodze/GitHubHowTo.git" from the github repo
  • copy and paste the second link "git push -u origin master " from the github repo

    • Congratulations You Are Finished!
